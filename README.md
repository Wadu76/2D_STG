


# 2D STG - 弹幕射击游戏

<p align="left">
  <img src="https://img.shields.io/badge/Unity-2022.x-blue" alt="Unity Version">
  <img src="https://img.shields.io/badge/C%23-10.0-purple" alt="C# Version">
 
</p>
本项目是在玩了飞吧龙骑士后即兴用AI快速开发的一个demo（大概3h开发时长）。

一个2D弹幕射击躲避游戏，使用Unity开发。玩家可以控制战机躲避弹幕、击败敌人、挑战BOSS，并通过升级获得强大的天赋能力。

---

## 🎮 游戏功能

### 玩家系统
- 键盘控制移动（WASD或方向键）
- 自动连续开火
- 玩家血量系统
- 属性加成系统（伤害、血量、移速、射速）

### 敌人系统
- 多种敌人类型不断刷新
- 敌人AI移动和射击
- 敌人血量和伤害系统
- 波数递进系统

### BOSS系统
- 每10波生成一次BOSS
- 散弹攻击（60度扇形三发）
- 追踪子弹攻击（追踪玩家，10秒后消失）
- BOSS血量是普通敌人的5倍

### 天赋系统
| 天赋 | 解锁等级 | 效果 |
|------|----------|------|
| 伤害提升 | 1级+ | 增加子弹伤害 |
| 血量提升 | 1级+ | 增加最大血量 |
| 移速提升 | 1级+ | 增加移动速度 |
| 射速提升 | 1级+ | 增加射击频率 |
| 寒冰子弹 | 5级+ | 5%几率冰冻敌人1秒 |
| 散弹模式 | 10级+ | 同时发射3发子弹 |

### 等级计算公式

| 等级 | 所需经验 | 累计经验 |
|------|----------|----------|
| 1→2 | 150 | 150 |
| 2→3 | 200 | 350 |
| 3→4 | 250 | 600 |
| 4→5 | 300 | 900 |
| 5→6 | 350 | 1250 |
| ... | ... | ... |

**经验公式**：
```
升级所需经验 = 100 + (当前等级 × 50)
```

### 属性加成公式

每次选择天赋时，对应属性会按照以下公式增加：

| 属性 | 加成公式 | 说明 |
|------|----------|------|
| 伤害 | `伤害倍率 = 1.0 + (选择次数 × 0.1)` | 每次+10%伤害 |
| 血量 | `血量倍率 = 1.0 + (选择次数 × 0.1)` | 每次+10%血量 |
| 移速 | `移速倍率 = 1.0 + (选择次数 × 0.05)` | 每次+5%移速 |
| 射速 | `射速倍率 = 1.0 + (选择次数 × 0.1)` | 每次+10%射速 |
| 寒冰概率 | `冰冻概率 = 0% + (选择次数 × 5%)` | 每次+5%冰冻几率 |

### 示例计算

- **5级时冰冻概率**：选择1次寒冰子弹 = 5%
- **10级时冰冻概率**：选择1次寒冰子弹 = 5%（不会再自动增加）
- **10级散弹伤害**：每发子弹 = 基础伤害 × 0.7（70%）

```

│   │   ├── PlayerShooter.cs        # 玩家射击系统
│   │   └── PlayerHealth.cs        # 玩家血量管理
│   │
│   ├── Enemy/
│   │   ├── EnemyController.cs     # 敌人AI和移动
│   │   ├── EnemyHealth.cs         # 敌人血量管理

│   │
│   ├── Boss/
│   │   ├── BossController.cs           # BOSS控制逻辑
│   │   ├── BossHealth.cs               # BOSS血量管理
│   │   └── HomingBulletController.cs   # 追踪子弹控制
│   │
│   ├── UI/
│   │   ├── UIManager.cs              # 分数和波数显示
│   │   ├── LevelUpUIManager.cs       # 升级面板管理
│   │   └── PauseMenuManager.cs       # 暂停菜单管理
│   │
│   ├── Game/
│   │   ├── LevelManager.cs           # 等级和天赋系统
│   │   └── ScoreManager.cs           # 分数管理
│   │
│   └── Core/
│       ├── BulletController.cs        # 子弹控制
│       └── ObjectPool.cs             # 对象池
│
├── Scenes/                    # 游戏场景
├── Prefabs/                  # 预制体
├── Sprites/                  # 精灵图片
└── TextMesh Pro/            # UI字体资源
```

## 🚀 安装和运行

1. 克隆仓库到本地
   ```bash
   git clone https://github.com/Wadu76/2D_STG.git
   ```

2. 使用Unity Hub打开项目
   - 打开Unity Hub
   - 点击 "Open" -> 选择克隆的项目文件夹

3. 打开场景并运行
   - 打开 `Assets/Scenes/` 下的游戏场景
   - 点击Play按钮运行游戏
